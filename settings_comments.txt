WindowSize:

Width: Ширина окна. 0 = размер по умолчанию.

Hight: Высота окна. 0 = размер по умолчанию. (Опечатка: должно быть "Height".)

Bot:

IsWhiteBot: Является ли игрок за белые фигуры ботом (false = человек).

IsBlackBot: Является ли игрок за черные фигуры ботом (true = бот).

WhiteBotLevel: Уровень сложности бота для белых фигур. 0 = бот отключен.

BlackBotLevel: Уровень сложности бота для черных фигур. 5 = сложный.

BotScoringType: Как бот оценивает ходы (например, учитывает количество фигур и их потенциал).

BotDelayMS: Задержка перед ходом бота (в миллисекундах). 0 = без задержки.

NoRandom: Если true, бот делает детерминированные ходы (без случайности).

Optimization: Уровень оптимизации бота (O1 = базовый, O2/O3 = более продвинутый).

Game:

MaxNumTurns: Максимальное количество ходов в игре. Если превышено, игра завершается.
